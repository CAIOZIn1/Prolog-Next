## Sobre o projeto

Projeto desenvolvido em Next e Prolog

## Rodar o projeto

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

> Obs.: É necessário ter instaladdo Node e SWI Prolog
